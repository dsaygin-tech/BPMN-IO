# BPMN-IO

Десктопный редактор BPMN для Windows и macOS. Построен на [bpmn-js](https://bpmn.io) с симуляцией токена, панелью свойств и экспортом в файлы `.bpmn`.

## Возможности

- Редактирование диаграмм BPMN 2.0 (задачи, шлюзы, события, дорожки и др.)
- **Валидация BPMN (bpmnlint)** — проверка диаграммы на типичные ошибки моделирования
- **Сетка на холсте** — визуальная точечная сетка для выравнивания элементов
- **Мини-карта** — обзор всей диаграммы и быстрая навигация по холсту
- **Симуляция токена** — анимация движения токена по процессу
- Открытие, сохранение и экспорт диаграмм
- Открытие файлов через диалог, перетаскивание в окно или двойной клик по `.bpmn`
- Экспорт в **BPMN (.bpmn)**, **JSON (.json)**, **SVG (.svg)**, **PNG (.png)** и **PDF (.pdf)**
- Панель свойств для настройки элементов

## Требования

- Node.js 20+
- npm

## Разработка

```bash
nvm use
npm install
npm run dev
```

Запускаются Vite и Electron. При изменении кода интерфейса приложение перезагружается автоматически.

## Сборка установщиков

```bash
# macOS (.dmg, .zip)
npm run dist:mac

# Windows (.exe-установщик, portable)
npm run dist:win

# Текущая платформа
npm run dist
```

Установщики сохраняются в папку `release/`.

### Сборка Windows x64 (не ARM)

```bash
# Windows x64
npm run dist:win -- --x64

# Альтернатива напрямую через electron-builder
npm run build && electron-builder --win --x64
```

Если вы собираете на macOS и NSIS-установщик не собирается, используйте portable-сборку:

```bash
npx electron-builder --win --x64 --config.win.target=portable
```

## Горячие клавиши

| Действие | Сочетание |
|----------|-----------|
| Новый | `Ctrl/Cmd + N` |
| Открыть | `Ctrl/Cmd + O` |
| Сохранить | `Ctrl/Cmd + S` |
| Сохранить как | `Ctrl/Cmd + Shift + S` |
| Симуляция токена | `Ctrl/Cmd + Shift + T` |
| Пауза симуляции | `Space` (в режиме симуляции) |
| Сброс симуляции | `R` (в режиме симуляции) |
| Журнал симуляции | `L` (в режиме симуляции) |

## Симуляция токена

Нажмите **Token Simulation** на панели холста (или `Ctrl/Cmd + Shift + T`). По умолчанию токены **запускаются сразу** (режим **Auto-start**).

### Режимы запуска (видны в режиме симуляции)

| Элемент | Описание |
|---------|----------|
| **Start: Auto-start** *(по умолчанию)* | Токены стартуют сразу при входе в симуляцию или при переключении на этот режим |
| **Start: Interactive** | Токены ждут, пока вы нажмёте ▶ на Start Event |
| **Step mode** | Токен останавливается на каждой задаче |

### Управление во время симуляции

Pause, reset, log и скорость — через встроенную палитру bpmn-js-token-simulation на холсте и горячие клавиши:

| Действие | Клавиша |
|----------|---------|
| Пауза / продолжение | `Space` |
| Сброс | `R` |
| Журнал симуляции | `L` |
| Выход из симуляции | `Ctrl/Cmd + Shift + T` |

Нажмите **Exit Simulation**, чтобы вернуться к редактированию.

## Валидация BPMN (bpmnlint)

Редактор проверяет диаграмму по набору правил [bpmnlint](https://github.com/bpmn-io/bpmnlint) через интеграцию [bpmn-js-bpmnlint](https://github.com/bpmn-io/bpmn-js-bpmnlint).

1. На панели холста нажмите кнопку **Lint** (иконка галочки) — проверка включается и обновляется при каждом изменении.
2. Элементы с проблемами помечаются значками **ошибки** (красный) или **предупреждения** (жёлтый); наведите курсор, чтобы увидеть описание.
3. Повторное нажатие **Lint** отключает подсветку.

По умолчанию используется пресет `bpmnlint:recommended` (файл `src/.bpmnlintrc`): отсутствие start/end event, изолированные элементы, пустые подписи, некорректные потоки и другие типичные ошибки. Правила и их уровень (`error` / `warn`) можно изменить в `.bpmnlintrc`, после чего выполните `npm run pack:bpmnlint` (скрипт также запускается автоматически перед `npm run dev` и `npm run build`).

## Экспорт

Экспорт доступен через кнопку **Export** на панели инструментов или меню **File → Export**:

| Формат | Описание |
|--------|----------|
| `.bpmn` | BPMN 2.0 XML — исходная модель для редактирования |
| `.json` | JSON-представление BPMN-модели |
| `.svg` | Векторное изображение диаграммы |
| `.png` | Растровое изображение диаграммы |
| `.pdf` | PDF-документ с векторной диаграммой |
| `.png` (simulation) | Статичный снимок диаграммы с токенами и подсветкой |
| `.webm` (simulation) | **Видео WebM** — анимация движения токенов (VLC, браузеры) |

### Экспорт анимации симуляции

1. Нажмите **Export Animation** (или **Simulation WebM** в меню Export)
2. Настройте режим записи и нажмите **Start export**
3. Симуляция **запустится автоматически** с начала процесса
4. Запись **остановится сама**, когда все токены дойдут до конца

**WebM** — видео с движущимися токенами (VLC, браузеры).

Статичный **Simulation PNG** сохраняет один кадр (нужно включить симуляцию заранее).
